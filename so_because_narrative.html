Linking past events using So and Because

BECAUSE (The Reason)

Usamos because para explicar el porqué de algo. Nos da la razón o causa de la acción principal.
"I stayed at home because it was raining heavily."
(Razón: Estaba lloviendo).

SO (The Result)

Usamos so para mostrar el resultado o la consecuencia de una acción previa. Suele llevar una coma antes.
"It was raining heavily, so I stayed at home."
(Resultado: Me quedé en casa).

💡 The Golden Flip

Recuerda que son opuestos. Si cambias el orden de la oración, debes cambiar la palabra.

"I lost my passport because I was careless."
is the exact same as...
"I was careless, so I lost my passport."
